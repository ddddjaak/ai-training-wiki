# Claude Code 芯片工程师 Wiki

## 这是什么

一套面向**芯片工程师**（嵌入式、AE、SE）的 Claude Code 学习资料。从安装配置到日常工作流的深度覆盖——聚焦你每天最花时间的那些环节。

不是通用 AI 教程。所有示例、场景、案例都是芯片工程师的真实工作——寄存器调试、客户方案交付、需求分解、设计审查。

## 我该从哪里开始

**所有人先看 M1**（工具上手，约 60 分钟）。装好 Claude Code + CC Switch，学会基础对话和命令。

然后按你的角色选路径：

| 我是 | 学完 M1+M2 后看 | 核心内容 |
|------|----------------|---------|
| 嵌入式工程师 / AE | **M3** AE 实战 | 调试固件 bug、代码审查、TDD、客户方案交付（基于 ae-skills 插件） |
| 系统工程师 | **M4** SE 实战 | 需求分解、架构设计、设计审查、追溯矩阵、发布审查 |

M2 建议所有人都过一遍——讲的是 Claude Code 的高级能力（skill 体系、MCP 插件、hooks、并行代理），是后面所有角色模块的基础。

## 耗时预估

| 模块 | 页面数 | 预计耗时 |
|------|--------|---------|

每个页面独立可用——你可以按顺序学，也可以挑你当前最需要的那个场景直接看。

## 环境要求

- **Claude Code CLI**（通过 npm 安装）
- **CC Switch**（公司 API Key 管理工具——不是个人 Anthropic 账号）
- **ae-skills 插件**（V1.0+，嵌入式/AE 角色核心依赖）
- Windows / macOS / Linux 均可（Windows 需 Git Bash 或 WSL）

详见 M1-1 安装与启动。

## 阅读方式

每个页面都设计为独立可读——前置条件在页面顶部标注了，不需要按顺序翻。遇到截图缺失的地方，页面上会有描述性占位符告诉你"这里应该看到什么"，不影响理解。
